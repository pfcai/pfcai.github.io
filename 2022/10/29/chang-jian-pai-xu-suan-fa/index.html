今天把常见的排序算法进行总结，包括时空复杂度和一些特殊情况下的优劣。

low三人

冒泡排序

冒泡排序是从数组元素第一个开始，两两比较，循环n-1次，每一次循环都会确定一个值的位置。

public void bubbleSort(intp[] list) {
  int len = list.length;
  for (int i = 0; i < len - 1; i++) {
    boolean flag = false;
    for (int j = 0; j < len - i; j++) {
      if (list[j] > list[j + 1]) {
        int temp = list[j + 1];
        list[j + 1] = list[j];
        list[j] = temp;
        flag = true;
      }
    }
    if (flag == false) {
     return; 
    }
  }
}

冒泡排序的时间复杂度是O(n2),最坏的情况是数组有序，所以可以进行优化，添加标识为判断是否存在交换，不存在交换则结束循环。

选择排序

每次选择一个数组中最小的数放入数组，所以外层循环n次。

public void selectSort(int[] list) {
  for (int i = 0; i < list.length; i++){
    int min = i;//min是最小那个数的索引
    for (int j = i + 1; j < list.length; j++) {
      if (list[min] > list[j]) {
        min = j;
      }
    }
    int temp = list[min];
    list[min] = list[i];
    list[i] = temp;
  }
}

时间复杂度是O(n2)，

插入排序

假设插入排序就是抽牌理牌的过程，手中的牌是有序的，每次抽一张牌将它插到手中并保持有序。

public void insertSort(int[] list) {
  for (int i = 1;i < list.length; i++) {
    int j = i - 1;
    int temp = list[i];
    while (j > 0 && list[j] > temp) {
      //把大牌向后调整
      list[j + 1] = list[j];
      j--;
    }
    list[j] = temp;
  }
}

插入排序的时间复杂度也是O(n2)，因为每次需要移动牌，从1 + 2 + … + n = n(n + 1)。

牛三人

快速排序

快速排序的中心思想是取一个元素p（第一个元素），使元素p归位，列表被p分成两部分，左边都比p小，右边比p大；递归完成排序。

public void fastSort(int[] list, int left, int right) {
  if (left < right) {
    int mid = partition(list, left, right);
    fastSort(list, left, mid - 1);
    fastSort(list, mid + 1, right);
  }
}

public void partition(int[] list, int left, int right) {
  int temp = list[left];
  while (left < right) {
    while (left < right && list[left] <= temp) {
      left++;
    }
    list[right] = list[left];
    while (left < right && list[right] >= temp) {
      right--;
    }
    list[left] = list[right];
  }
  list[left] = temp;
  return left;
}

快速排序的时间复杂度是O(nlogn)，快速排序存在最坏情况，也就是数组为倒序

归并排序

归并排序的思路是，将数组分成两部分，分别将左边和右边变有序，然后使用将两个有序数组合并为一个有序数组的方法

public void mergeSort(int[] list, int left, int right, int[] temp) {
  if (left < right) {
    int mid = (left + right) / 2;
    mergeSort(list, left, mid, temp);//左有序
    mergeSort(list, mid + 1, right, temp);//右有序
    merge(list, left, mid, right, temp);
  }
}

public void merger(int[] list, int left, int mid, int right, int[] temp) {
  int i = left;
  int j = mid + 1;
  int count = 0;
  while (i <= mid && j <= right) {
    if (list[i] > list[j]) {
      temp[count++] = list[j++];
    } else {
      temp[count++] = list[i++];
    }
  }
  while (i <= mid) {
    temp[count++] = list[i++];
  }
  while (j <= right) {
    temp[count++] = list[j++];
  }
  //将排好的顺序放入原数组中
  count = 0;
  int leftI = left;
  while (leftI <= right) {
    list[left++] = temp[count++];
  }
}

归并排序的时间复杂度：O(nlogn)，归并有logn层，每一层循环n次。

堆排序

理解堆排序需要知道二叉树的一些相关知识，这里不讨论。堆排序分为大根堆和小根堆。顾名思义，大根堆是父节点元素一定比子节点大，小根堆则相反。堆排序我们首先需要根据数组构建出堆，然后进行调堆、出数的操作。二叉树有两种存储方式，一种是数组（本写法默认数组存放），一种是链式存储。

public void sift(int[] list, int low, int high) {
  int i = low;//表示根节点
  int j = i * 2 + 1;//表示左孩子
  int temp = list[i];
  while (j <= high) {
    if (j + 1 <= high && list[j + 1] > list[j]) {//右孩子大
      j = j + 1;
    }
    if (list[j] > temp) {//孩子节点比父节点大，交换节点，更新标识位
      list[i] = list[j];
      i = j;
      j = i * 2 + 1;
    } else {
      	list[i] = temp;//可省略
      	break;
    }
  }
  list[i] = temp;
}

public void heapSort(int[] list) {
  //从最后一个父节点向上建堆
  for (int i = (list.length - 2) / 2; i > -1; i--) {//最后一个元素是len-1，子节点->父节点公式：“(i - 1) / 2”
    sift(list, i, list.length - 1);
  }
  //出数-不额外申请空间
  for (int i = list.length - 1; i > -1; i--) {
    int temp = list[0];
    list[0] = list[i];
    list[i] = temp;
    sift(list, 0, i - 1);
  }
}

堆排序的时间复杂度：sift函数是O(logn)，建堆为O(n),出数为O(n)，所以是2 * O(n*logn) = O(nlogn)。但快速排序比堆排序快一点。

其他类型排序

希尔排序

希尔排序是分组插入排序算法，通过选取一个gap，将每组相邻距离为gap的元素进行直接插入排序，每进行完一次排序，gap的变为原来的一半，直到gap为1为止。希尔排序每趟不使元素有序，而是使整体数据越接近有序

public void insertShellSort(int[] list, int gap) {
  for (int i = gap; i < list.length; i++) {
    int j = i - gap;
    int cur = list[i];
    while (j >= 0 && list[j] > cur) {
      list[j + gap] = list[j];
      j -= gap;

public void shellSort(int[] list) {
  int gap = list.length / 2;
  while (gap > = 1) {
    insertShellSort(list, gap);
    gap /= 2;

希尔排序的时间复杂度与选取的gap序列有关，不同的gap时间复杂度不同，可以在wiki上看到现存gap的时间复杂度。

桶排序

首先将列表元素分在不同的桶中，在将桶中的元素排序。比较关键的是桶的大小和桶的个数。

public void binSort(int[] list, int bucketSize) {
  //首先计算一下桶的个数
  int maxNum = list[0], minNum = list[0];
  for (int i = 0; i < list.length; i++) {
    maxNum = Math.max(list[i], maxNum);
    minNum = Math.min(list[i], minNum);
  }
  List<List<Integer>> res = new ArrayList<>();//使用list模拟桶
  //创建桶
  int bucketNum = (maxNum - minNum) / bucketSize;
  for (int i = 0;i < bucketNum; i++) {
    res.add(new ArrayList<>());
  }
  //将数依次放入桶中
  for (int i = 0;i < list.length; i++) {
    res.get((list[i] - minNum) / bucketSize)).add(list[i]);
  }
  //对每个桶进行排序
  for (int i = 0; i < bucketNum; i++) {
    Collections.sort(res.get(i));
  }
  //将桶中的数据放入数组
  int index = 0;
  for(int i = 0; i < bucketNum; i++) {
    for (int j = 0; j < res.get(i).size(); j++) {
      list[index++] = res.get(i).get(j);
    }
  }
}

桶排序的时间复杂度和数据分布有关系，对不同的数据分布采取不同的分桶策略。（k表示一个桶平均的长度）平均情况下时间复杂度为O(n + k)，最坏情况为：O(n2k)。空间复杂度是O(nk)。

计数排序

计数排序非常好理解，创建一个计数数组，将元素依次放入计数数组中，每放入一个数，计数数组值+1，最后遍历计数数组打印元素。因为数组本身是有序的，所以可以进行排序。

public void countSort(int[] list) {
  //找出最大的值目的是创建数组
  int max = list[0];
  for (int i : list) {
    max = Math.max(i,max);
  }
  int[] res = new int[max];
  for (int i : list) {
    res[i]++;
  }
  for (int i = 0; i < res.length; i++) {
    for (int j = 0; j < res[i]; j++) {
      System.out.println(i);
    }
  }
}

计数排序时间复杂度是O(n)，但是有限制，即n不能太大。

基数排序

基数排序的原理是先从个位排序，然后百位、千位…直到位数最大的长度为止。具体实现：创建10个桶表示每位数上的值0~9，依次取个位上的数放入桶中，然后将桶内元素排序并放回原数组，接着取十位、百位等等。

public void redixSort(int[] list) {
  int maxNum = list[0];
  for (int i : list) {
    maxNum = Math.max(maxNum,i);
  }
  int count = 0;//循环计数
  while (Math.pow(10, count) <= maxNum) {
    int bucketNum = 10;//0-9
    List<List<Integer>> res = new ArrayList<>();
    while (bucketNum > 0) {
      res.add(new ArrayList<>());
      bucketNum--;
    }
    //取每一位的值并放入桶中
    int digit = 0;
    for (int i : list) {
      digit = (int)(i / Math.pow(10,count)% 10 );
      res.get(digit).add(i);
    }
    //将桶中的数组放入数组
    int index = 0;
    for (int i = 0;i < bucketNum; i++) {
      for (int j = 0; j< res.get(i).size(); j++) {
        list[index++] = res.get(i).get(j);
      }
    }
    count++;//取下一位
  }
}

基数的时间复杂度是：O(nk)，k是最大数的数字位数。空间复杂度为O(k + n)。
