双链表

双链表
数据结构与算法

链表是一种通过指针串联在一起的线性结构，每一个节点由两部分组成，一个是数据域一个是指针域，最后一个节点的指针域指向null。

链表类型

  • 单链表
  • 双链表
  • 循环链表

链表存储方式

链表是通过指针域的指针链接在内存中各个节点。所以链表中的节点在内存中不是连续分布的 ，而是散乱分布在内存中的某地址上，分配机制取决于操作系统的内存管理。

链表的定义

链表由多个节点组成，一个节点有一个value。这里我们还需要自己写一个节点的构造方法，虽然不写系统会生成默认的构造函数，但为了减少出错还是需要自己定义。

class ListNode {
  int val;
  ListNode next;
  
  ListNode() {}
  
  ListNode(int val) {
    this.val = val;
  }
}

链表例题

移除链表元素

leetcode203

题意：删除链表中等于给定值 val 的所有节点。

示例 1：
输入：head = [1,2,6,3,4,5,6], val = 6
输出：[1,2,3,4,5]

思路

使用双指针进行判断并且删除目标元素。使用虚拟头节点简化操作。

public ListNode removeElements(ListNode head, int val) {
  if (head == null) return head;
  ListNode newHead = new ListNode(-1);
  newHead.next = head;
  ListNode front = newHead;
  ListNode cur = head;
  while (cur != null) {
   if (cur.val == val) {
     front.next = cur.next;
   } else {
			front = cur;
   }
    cur = cur.next;
  }
  return newHead.next;
}

设计链表

leetcode707

在链表类中实现这些功能：

  • get(index)：获取链表中第 index 个节点的值。如果索引无效，则返回-1。
  • addAtHead(val)：在链表的第一个元素之前添加一个值为 val 的节点。插入后，新节点将成为链表的第一个节点。
  • addAtTail(val)：将值为 val 的节点追加到链表的最后一个元素。
  • addAtIndex(index,val)：在链表中的第 index 个节点之前添加值为 val 的节点。如果 index 等于链表的长度，则该节点将附加到链表的末尾。如果 index 大于链表长度，则不会插入节点。如果index小于0，则在头部插入节点。
  • deleteAtIndex(index)：如果索引 index 有效，则删除链表中的第 index 个节点。

思路

就是自己定义结构体并且实现增删改查的功能。还是需要使用虚拟头节点去简化操作，并且需要一个size参数记录节点的个数，一个head当虚拟的头节点。

public class MyLinkedList {
    int size;
    ListNode head;//虚拟头节点

    public MyLinkedList() {
        size = 0;
        head = new ListNode(0);//虚拟头节点
    }

    public int get(int index) {
        if (index < 0 || index >= size) {
            return -1;
        }
        ListNode cur = head;
        for (int i = 0; i <= index; i++) {
            cur = cur.next;
        }
        return cur.val;
    }

    public void addAtHead(int val) {
        addAtIndex(0, val);
    }

    public void addAtTail(int val) {
        addAtIndex(size, val);
    }

    public void addAtIndex(int index, int val) {
        if (index > size) {
            return;
        }
        if (index < 0) {
            index = 0;
        }
        size++;
        ListNode cur = head;
        for (int i = 0; i < index; i++) {
            cur = cur.next;
        }
        ListNode newNode = new ListNode(val);
        newNode.next = cur.next;
        cur.next = newNode;
    }

    public void deleteAtIndex(int index) {
        if (index < 0 || index >= size) {
            return;
        }
        size--;//位置忽略了，只要是参数合法就立刻减一
        if (index == 0) {
            head = head.next;
            return;
        }

        ListNode cur = head;
        for (int i = 0; i < index; i++) {
            cur = cur.next;
        }
        cur.next = cur.next.next;
    }
}

反转链表

leetcode206

题意：反转一个单链表。

示例: 输入: 1->2->3->4->5->NULL 输出: 5->4->3->2->1->NULL

思路

这道题需要注意的就是在交换的时候一定要保存好下一个节点，可以使用双指针和递归。

双指针

public ListNode reverseList(ListNode head) {
  ListNode pre = null;
  ListNode cur = head;
  ListNode tmp;//保存节点
  while (cur != null) {
    tmp = cur.next;
    cur.next = pre;
    pre = cur;
    cur = tmp;
  }
  return pre;
}

递归

public ListNode reverseList(ListNode head) {
  return reverse(head, null);
}

public ListNode reverse(ListNode cur, ListNode pre) {
  if (cur == null) return pre;
  ListNode tmp = cur.next;
  cur.next = pre;
  return reverse(tmp, cur);
}

两两交换链表种的元素

leetcode24

给定一个链表，两两交换其中相邻的节点，并返回交换后的链表。

思路

就是单纯的交换两两元素，做题的时候一定要画图这样能理清思路。使用虚拟头节点优化代码。

public ListNode swapPairs(ListNode head) {
  ListNode newHead = new ListNode(-1);
  newHead.next = null;
  ListNode cur = newHead;
  while (cur.next != null && cur.next.next != null) {
    ListNode tmp = head.next.next;
    cur.next = head.next;
    head.next.next = head;
    head.next = tmp;
    cur = head;
    head = head.next;
  }
  return newHead.next;
}

删除链表的第倒数第n个元素

leetcode19

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

思路

使用双指针，先让快指针走n步，然后快慢指针同时走，当快指针到队尾的时候，慢指针的下一个就是倒数第n个元素。

public ListNode removeNthFromEnd(ListNode head, int n) {
  ListNode newHead = new ListNode(-1);
  newHead.next = head;
  ListNode fast = newHead;
  ListNode slow = newHead;
  //快指针先走
  for (int i = 0; i < n; i++) {
    fast = fast.next;
  }
  //一起走
  while (fast.next != null) {
    fast = fast.next;
    slow = slow.next;
  }
  //删除
  slow.next = slow.next.next;
  return newHead.next;
}

链表相交

leetcode链接

给你两个单链表的头节点 headA 和 headB ，请你找出并返回两个单链表相交的起始节点。如果两个链表没有交点，返回 null 。

思路

遍历链表得到两个链表的长度，在长链表中遍历直到剩余长度等于短的链表长度，然后一起遍历，找到相同的节点。没找到返回null。

public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
  ListNode curA = headA;
  ListNode curB = headB;
  int lenA = 0, lenB = 0;
  //遍历求长度
  while (curA != null) {
    lenA++;
  }
  while (curB != null) {
    lenB++;
  }
  //重置指针
  curA = headA;
  curB = headB;
  //使A结尾的是最大的
  if (lenB > lenA) {
    ListNode tmp = curA;
    curA = curB;
    curB = tmp;
    int tm = lenA;
    lenA = lenB;
    lenB = tm;
  }
  //移动到和短的那个链表同样长度的地方
  int gap = lenA - lenB;
  for (int i = 0; i < gap; i++) {
    curA = curA.next;
  }
  //循环判断剩下的节点
  while (curA != null) {
    if (curA == curB) {
      return curA;
    }
    curA = curA.next;
    curB = curB.next;
  }
  return null;
}

环形链表

leetcode142

题意： 给定一个链表，返回链表开始入环的第一个节点。 如果链表无环，则返回 null。为了表示给定链表中的环，使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。

思路

判断是否有环还是使用双指针，快指针每次走两步，慢指针每次走一步。如果有环则必定在环内相遇。判断环的起点也是用两个指针，一个从头节点开始一个从相遇节点开始。当两指针相遇时的节点就是环的入口。具体思路看这里。

public ListNode detectCycle(ListNode head) {
  //定义快慢指针
  ListNode fast = head;
  ListNode slow = head;
  //循环判断是否有环
  while (fast.next != null && fast.next.next != null) {
    fast = fast.next.next;
    slow = slow.next;
    if (slow == fast) {
      //有环找入口
      ListNode index1 = head;
      ListNode index2 = slow;
      while (index1 != index2) {
        index1 = index1.next;
        index2 = index2.next;
      }
      return index1;
    }
  }
  return null;
}
